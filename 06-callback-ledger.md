## 0. Why this document exists

This file prevents the novel from becoming either:

1. a chain of isolated atmospheric scenes, or
2. a mystery that withholds too much and pays off too little.

Every important clue, image, phrase, object, emotional pattern or transition seam should either:

- return with changed meaning,
- pay off in a later reveal,
- or be deliberately marked as decorative and removed.

This ledger is not prose. It is the memory of the manuscript.

## 8. Anti-drift watchlist

If any chapter harvest shows these patterns, flag immediately and repair before continuing.

| Risk | Symptom | Fix |
|---|---|---|
| All-dead drift | Passengers speak / feel like ghosts waiting for judgment. | Restore living psychological stakes; the train is not afterlife. |
| Generic guilt train | Clues become abstract trauma symbols unrelated to Albula 2008. | Re-anchor in route, documents, objects, weather, railway process. |
| Schaffner exposition | Mancini explains rules in paragraphs. | Cut to one cold line + scene evidence. |
| Levin leak | Levin internally names his hidden role before reveal. | Replace with body reaction, avoidance, or sensory fragment. |
| Marlene romance shrink | Marlene exists mainly to comfort Levin. | Give her her own observation, resistance, memory path and cost. |
| Motif inflation | Schnee / 02:47 / coffee / camera all appear in every scene. | Let each chapter carry 1–3 motifs only. |
| Simplified morality | Disappearances feel like simple sentencing. | Reframe as recognition / refusal mechanics and human consequences. |
| Age / timeline drift | Levin 24 / Marlene 26 in present reappears. | Correct to winter 2025/26: Levin 36/37, Marlene 43/44. |
| Seam neglect | Anika, Selma/Eliah or Marlene-aftershock vanish between their structural beats. | Check §6 before each relevant beat plan. |

## 9. Update rule

After every chapter:

1. Add chapter-specific seeds to §7.
2. Update statuses in §2–§6.
3. Mark any overused motif in §7 watchlist.
4. If a planned seed no longer serves the book, mark it `✂ cut` instead of silently forgetting it.
5. If a new recurring pattern emerges, add it to §3, §6 or §8.

Do not let this file become decorative. If it is not updated, the mystery will drift.
